# Changelog / 更新日志

All notable changes are documented here. 重要变更记录在此。

## 1.0.0 - 2026-07-15

### Added / 新增

- Complete create → invite → join → submit → review → layout → PDF workflow / 完整的创建、邀请、加入、投稿、审核、排版与 PDF 闭环
- Persisted book templates, layouts, uploads, and publication order / 持久化书籍模板、排版、上传文件与出版顺序
- Real generated and imported article-number pools / 真实的自动生成与文件导入文章编号池
- English and Simplified Chinese interfaces with light/dark themes / 中英文界面与浅色、深色主题
- Route, hero, dialog, loading, and drag interaction animation with reduced-motion support / 页面、首页、弹窗、加载与拖拽动效，并支持减少动态效果
- Frontend unit tests, backend integration tests, CI, production containers, and release documentation / 前端单元测试、后端集成测试、CI、生产容器与发布文档

### Security / 安全

- Upload type, size, content, filename, and storage-path validation / 上传类型、大小、内容、文件名与存储路径校验
- Local/private-LAN deployment boundary documented / 明确本机与私有局域网部署边界
